遊戲規則
準則說明
Connect準則:己方所有符號的前，後，左，右，右前，左前，右後，右左方，單位為一格，其餘無法設置
Blocker準則:用四枚符號形成矩形，我方形成指定形狀時，其中對方是無法設置的
Divider準則:己方所有符號的前，後，左，右，右前，左前，右後，右左方，單位為無限距離長，除非該方有非我方符號存在，其餘無法設置
Adapter準則:即己方所有符號的右前前，左前前，右右前，左左前，右後後，左後後，右右後，左左後方，單位為ㄧ格，其餘無法設置
名稱說明
封限區:O為此色，X為此色
  1. 當對方封限區有一枚我方符號時，我方獲勝
  2. 當我方封限區皆為我方符號時，對方獲勝
封區:O為此色，X為此色
  1. 當我方限區皆為我方符號時，對方獲勝
殭屍:皆為此色，字樣為"Z"
障礙:皆為此色，字樣為" "
Square.Connect
規則說明
  1. 雙方第一回合皆可隨意設置一枚符號於場上
  2. 第二回合後設置符號需依照Connect準則設置
Connect:Attack
規則說明
  1. 場上會有固定的封區與封限區
  2. 雙方第一回合限定設置於指定區域
  3. 第二回合後設置符號需依照Connect準則設置
Connect:Defend
規則說明
  1. 雙方第一回合皆可隨意設置一枚符號於場上
  2. 第二回合後設置符號需依照Connect準則設置
  3. 在一區域佔下我方符號五枚，且無對方符號時，我方獲勝
Connect:Scheme
規則說明
  1. 場上會有隨機的的障礙
  2. 雙方第一回合皆可隨意設置一枚符號於場上，但不得設置於C3-G7的座標上
  3. 第一回合設置後會自動於附近產生封限區
  4. 第二回合後設置符號需依照Connect準則設置
Square.Blocker
規則說明
  1. 雙方皆可隨意設置一枚符號於場上
  2. 第二回合後設置符號需依照Blocker準則設置
Blocker:ByLine
規則說明
  1. 雙方皆可隨意設置一枚符號於場上
  2. 第二回合後設置符號需依照Blocker準則設置
  3. 用四枚符號形成矩形
  4. 用兩枚符號形成直線
Blocker:Forbid
規則說明
  1. 雙方皆可隨意設置一枚符號於場上
  2. 第二回合後設置符號需依照Blocker準則設置
  3. 對方不可設置的地方，我方也無法設置
Square.Divider
規則說明
  1. 雙方第一回合皆可隨意設置一枚符號於場上
  2. 第二回合後設置符號需依照Divider準則設置
Divider:Zombie
規則說明
  1. 雙方第一回合皆可隨意設置一枚符號於場上
  2. 第二回合後設置符號需依照Divider準則設置
  3. 不同的符號靠近時雙方皆會變成殭屍
Divider:Follow
規則說明
  1. 雙方第一回合皆可隨意設置一枚符號於場上
  2. 第二回合後設置符號需依照Divider準則設置
  3. 變更原生準則，原依照己方所有符號，改為依照己方上回合符號
Square.Adapter
規則說明
  1. 雙方第一回合皆可隨意設置一枚符號於場上
  2. 第二回合後設置符號需依照Adapter準則設置
