Function / 常用功能

常用的一般功能:
  1. 點選格子或以鍵盤輸入座標(E5即按下E+5)來設置符號
  2. 滑鼠靠近空白處或按下]或[即可喚出選單
  3. 點選Clean或按下Del以清除棋盤
  4. 點選Undo或按下Ctrl+Z到上一步
  5. 點選Redo或按下Ctrl+Y到下一步
  6. 點選Goto或按下Ctrl+H到指定步
  7. 點選Addons或按下Ctrl+Alt做功能設定

Question / 常見問題

Q:遊戲模式無法正常載入?
A:本小組主要使用Opera作為測試瀏覽器，Chrome，Vivaldi也可以正常使用，而其他瀏覽器(例如:FireFox，Internet Explorer或Safari)可能部分顯示或部分功能會不正常，建議安裝前面所提到的瀏覽器。

Q:功能無法設定?
A:可能是功能間無法相容，因此會取消設定。建議調整部分的設定，會發生衝突的設定如下：

衝突設定項目設定名稱
不支援快速判定中級Blocker,Anomal,ByLine
不支援快速查詢 Blocker,Anomal,Invert,Forbid,Follow,ByLine
不支援Connect隨機設置Attack,Castle
固定設定項目設定名稱
開啟原生規則固定提升效能，快速判定固定為初級Connect
感染回合必須小於30，大於10Zombie
障礙數量固定為18Scheme
反射回合必須小於10，大於4Mirror
王國大小必須小於15，大於3Kingdom
保留符號必須小於10，大於5Fallen

Q:棋盤顯示或遊戲規則異常?
A:基本上，所有模式不會發生異常，除非為了提升效能而開啟的功能，例如：快速查詢，提升效能，可以試著關閉上述兩者設定再進行遊戲，倘若還是異常，可以使用設定項目回報錯誤，流程如下：先開啟設定選單(點按右上角的齒輪圖樣，或是將滑鼠靠近左側或右側空白處喚出選單，點選最下方的Addons，或按Ctrl+Alt或是長按棋盤右下角的Clean)勾選基本設定的最後一項回報錯誤即可。

About / 關於我們

開發動機

　　棋類遊戲因電玩遊戲而趨向沒落，我們藉9x9的棋盤設計多種遊戲規則，結合成一個新的棋盤遊戲網站，希望藉由這個迥然不同的棋盤遊戲網站再一次喚起大眾對於棋盤遊戲的熱忱。

採用技術

　　主要採用HTML5並使用GitHub Pages服務，線上對戰則應用了Google Firebase作為資料庫，主要使用JavaScript編寫而成。
